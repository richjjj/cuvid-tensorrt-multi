## 回调参数说明
### data内容——json格式
    - "cameraID"    int     必须
    - "freshTime"   string  否
    - "eventNum"    int     否
    - "events"      list [event]    是
        - "event"
            - "id"
            - "event"
            - "box"       list[float]
            - "entertime" string
            - "outtime"   string
            - "score"     float

## 交通事件 传入参数定义
```json
"cameraID": 0, // 必须
"uri":"rtsp://192.168.128.12",  // 必须
"events":[  // 必须。可为空
    {
        "eventName":"违停", // 必须。属于已经支持的事件类型
        "enable": true,    // 必须。是否启用。true或false
        "roi":[     // 非必须。感兴趣区域。可多个区域
            {
                "nums": 4,  // 坐标点数，2为line,>=3为polygon
                "points":{
                    "x1": x1, "y1": y1,
                    "x2": x2, "y2": y2,
                    "x3": x3, "y3": y3,
                    "x4": x4, "y4": y4,
                    ...
                }
            },
            {
                "nums":2,
                "points":{...}
            }
        ]
    },
    {
        "eventName":"变道",
        "enable":false
    }
]
```
目前支持的事件类型：
- 违停
- 变道
- 行人闯入
- 拥堵
